Ajouter un composant `FeedbackNudge` dans `ResultsPage.tsx` qui :

1. **Apparaît 3 secondes** après le chargement des résultats (useEffect avec setTimeout 3000ms), uniquement si `results.length > 0`
2. **Position** : bandeau fixe en bas de l'écran (fixed bottom-0, full width), z-50, au-dessus du footer
3. **Contenu** :
   - Texte court : "Aidez-nous à améliorer le matching — les pouces haut/bas sur chaque résultat prennent 5 secondes." (FR) / "Help us improve matching — thumbs up/down on each result takes 5 seconds." (EN)
   - Bouton X (icône X de lucide-react) à droite pour fermer
4. **Comportement** :
   - Fermer le bandeau : `setNudgeVisible(false)` — immédiat, sans animation complexe
   - Stocker la fermeture dans `sessionStorage.setItem('nudge_dismissed', '1')` pour éviter la réapparition sur re-render
   - Vérifier au mount : si `sessionStorage.getItem('nudge_dismissed')` vaut '1', ne jamais afficher
5. **Style** : utiliser `--mc-warn` comme background (jaune, visible), `--mc-bg` comme couleur texte, cohérent avec la beta banner en haut de la page

Implémentation dans le fichier :
- Ajouter state : `const [nudgeVisible, setNudgeVisible] = useState(false);`
- Ajouter useEffect après l'useEffect existant du tracking :
  ```typescript
  useEffect(() => {
    if (!results.length) return;
    if (sessionStorage.getItem('nudge_dismissed') === '1') return;
    const t = setTimeout(() => setNudgeVisible(true), 3000);
    return () => clearTimeout(t);
  }, [results.length]);
  ```
- Ajouter le JSX du bandeau juste avant le `<footer>` de fin :
  ```tsx
  {nudgeVisible && (
    <div className="fixed bottom-0 left-0 right-0 z-50 flex items-center justify-between gap-4 px-6 py-3 mc-mono text-xs uppercase tracking-widest"
      style={{ background: "var(--mc-warn)", color: "var(--mc-bg)" }}>
      <span>
        {language === "fr"
          ? "Aidez-nous : les pouces haut/bas sur chaque résultat prennent 5 secondes."
          : "Help us: thumbs up/down on each result takes 5 seconds."}
      </span>
      <button
        onClick={() => {
          setNudgeVisible(false);
          sessionStorage.setItem('nudge_dismissed', '1');
        }}
        className="flex-shrink-0 hover:opacity-70 transition-opacity"
        aria-label="Fermer"
      >
        <X className="w-4 h-4" />
      </button>
    </div>
  )}
  ```
- Ajouter `X` aux imports lucide-react existants en haut du fichier.

Ne pas modifier la logique existante de feedbackMutation ni les boutons pouce dans MatchCard.

La page admin utilise `x-admin-token` header pour les requêtes admin. Utiliser `useQuery` avec un `queryFn` custom (pas juste une URL string) pour passer le header :

```typescript
const adminToken = typeof window !== "undefined"
  ? localStorage.getItem("adminToken") || new URLSearchParams(window.location.search).get("admin_token") || ""
  : "";

const { data: dashboardData, isLoading: dashboardLoading } = useQuery({
  queryKey: ["/api/admin/feedback-dashboard", adminToken],
  queryFn: async () => {
    const res = await fetch("/api/admin/feedback-dashboard", {
      headers: { "x-admin-token": adminToken },
    });
    if (!res.ok) throw new Error("Non autorisé");
    return res.json();
  },
  enabled: !!adminToken,
});
```
