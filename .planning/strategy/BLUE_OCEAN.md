# SubventionMatch — Blue Ocean Strategy

> **Meta** — Date: 2026-04-18 · Framework: Kim & Mauborgne · Audience: agent IA (re-lecture future) · Langue: FR · Source concurrentielle: recherche web 2026-04-18 (voir `ANALYSE_CONCURRENCE.md` si créé).

---

## 0. TL;DR stratégique

- **Red ocean actuel** si positionnement = "matching subventions culture" → SubventIA déjà là à 120€/an, meilleure exécution.
- **Blue ocean accessible** en repositionnant sur : **"OS de financement du créatif indépendant"** (subventions + agents + galeries + labels + éditeurs + résidences + prix + bourses privées).
- **Cible blue ocean** : artiste/créatif individuel pro (musicien, écrivain, cinéaste, visuel) — catégorie orpheline entre "asso" (SubventIA) et "entreprise" (MAPi).
- **Modèle** : freemium (matching subvention gratuit) + tier pro payant (prospecting multi-canal) + B2B (labels/agents/écoles).
- **Score Blue Ocean actuel estimé : 4/10** — plan pour 10/10 en §7.

## 2. ERRC Grid — Four Actions Framework

### ÉLIMINER (ce que l'industrie fait mais sans valeur pour la cible créatif indépendant)

- **Couverture exhaustive multi-secteurs** (inutile pour un écrivain d'avoir aides agriculture)
- **Matching par entreprise/SIRET** (l'artiste indép. n'a pas toujours de structure juridique)
- **Jargon administratif** dans l'interface (DRAC, CNAP, CNC restent obscurs pour un débutant)
- **Nombre de dispositifs comme KPI marketing** ("11 700 aides !" crée de la friction, pas de la valeur)
- **Formulaires de profil longs** (remplacer par conversation IA qui déduit le profil)

### RÉDUIRE (sous le standard industrie)

- **Volume brut de dispositifs** : assumer ~500–800 de qualité > 11 000 bruités
- **Fonctionnalités B2B entreprise** (CA, effectifs, bilans) : inutile pour cible
- **Caution institutionnelle recherchée** : on n'a pas besoin d'être adoubé par CCI/Bpifrance
- **Pages de landing sectorielles SEO** : on vise communauté pro, pas trafic Google générique

### AUGMENTER (bien au-dessus du standard)

- **Qualité du matching sectoriel** (ton quality gate + prompt DeepSeek = déjà un edge, à doubler)
- **Profilage artiste par parcours/œuvre** (pas par formulaire admin) — style Spotify Wrapped
- **PDF dossier prêt-à-envoyer** personnalisé (ton Puppeteer existant, à hisser en différenciateur fort)
- **Suivi temporel des deadlines** + rappels (personne ne le fait bien)
- **Compréhension du "où en est l'artiste"** (émergent vs confirmé vs mi-carrière) → matching différent

### CRÉER (ce que l'industrie n'a jamais offert)

- **Catégorie produit "financement créatif 360°"** : subventions + bourses privées + résidences + prix + appels à projets agents/galeries/labels/éditeurs dans UNE plateforme
- **Prospecting privé** (agents, galeries, labels, éditeurs, producteurs) traité avec la même logique de matching que subvention publique — **vraie création de catégorie**
- **Score d'"adéquation carrière"** : pas juste "es-tu éligible" mais "est-ce que cette opportunité sert ton parcours ?"
- **Intelligence de calendrier annuel** : roadmap personnalisé des opportunités à préparer 3/6/12 mois à l'avance
- **Archive de dossiers réussis anonymisés** (preuve sociale + template → copie Y Combinator startup school)
- **Mode "collaboration"** : pour les collectifs / compagnies / groupes, plusieurs artistes partagent un dossier

## 3. Six Paths — opportunités blue ocean

### Path 1 — Industries alternatives

**Question** : qu'est-ce qu'un artiste utilise aujourd'hui à la place ?
- Excel perso avec liste d'appels
- Newsletters spécialisées (Quinzaine, Actualitté, devenir.art, Arty, Profession Spectacle)
- Groupes Facebook / Discord professionnels
- Agents / managers (pour les signés)
- Bouche-à-oreille entre pairs

**Valeur non satisfaite** : **curation personnalisée** (newsletters = one-size-fits-all) + **mémoire** (Excel se meurt) + **anticipation** (réseaux = réactifs).

→ **Action** : se positionner comme "l'agent digital de l'artiste indépendant", pas "le moteur de recherche de subventions".

### Path 2 — Groupes stratégiques

Clusters actuels :
- **Haut / institutionnel** : MAPi, Bpifrance services (B2B premium, cher, humain)
- **Milieu / associatif** : SubventIA (120€, multi-secteur, docs auto)
- **Bas / libre-service public** : Aides-Territoires, les-aides.fr (gratuit, UX admin)
- **Premium US** : Instrumentl, Granted (cher, anglo, NPO)

**Trou** : "premium artisan créatif" — qualité du matching + multi-rôles + prospecting privé. Pas gratuit, pas 120€, pas 300$/mo. Cible ~15–40€/mois.

### Path 3 — Chaîne des acheteurs

Chaîne actuelle : artiste paie (ou pas), utilise lui-même.

**Acheteurs alternatifs possibles** :
- **Écoles d'art / conservatoires / écoles de cinéma** (licence institutionnelle pour leurs étudiants/alumni)
- **Sociétés d'auteurs** (SACD, Sacem, SGDL, ADAGP) : pourraient offrir l'outil à leurs membres
- **Maisons de production / labels indépendants** : outil pour leurs artistes signés
- **Collectivités** (DRAC, régions) : financer l'accès pour leurs territoires

→ **Action blue ocean** : ne pas vendre qu'au B2C artiste, explorer **B2B2C via institutions** (modèle très différent, payeur ≠ user).

### Path 4 — Produits/services complémentaires

Avant/pendant/après demande de subvention :
- **Avant** : décider quoi demander, cadrer le projet, construire un budget
- **Pendant** : rédaction, relecture, preuves, justificatifs
- **Après** : suivi de décision, conventions, rapports d'activité, renouvellement, bilans

**Personne ne couvre le "après"**. Gros pain point : rapports d'activité, justificatifs, renouvellements ratés faute de mémoire.

→ **Action** : ajouter un module "suivi post-attribution" (rappels rapport, export bilan, template renouvellement).

### Path 5 — Appel fonctionnel vs émotionnel

Industrie actuelle = 100% fonctionnelle (trouver → matcher → documenter).

**Dimension émotionnelle manquante** : la demande de subvention est vécue comme **angoissante, injuste, opaque, décourageante**. Un artiste refuse des opportunités par **fatigue administrative**, pas par inéligibilité.

→ **Action** : ajouter dimension émotionnelle — ton bienveillant, messages de confiance, célébration des obtentions, normalisation des refus ("tu as postulé à X, c'est déjà un travail"). Anti-anxiété administrative. Différenciateur fort et non-imitable vite.

### Path 6 — Tendances temporelles

Tendances irréversibles à 3–5 ans :
1. **IA générative** : rédaction de dossier banalisée → la valeur se déplace vers **quoi demander** et **stratégie de portefeuille**, pas **comment rédiger**.
2. **Précarisation des intermittents / auteurs** : demande croissante de diversification des revenus.
3. **Fragmentation des carrières créatives** (musicien-producteur-TikTokeur) → besoin de matching multi-rôles natif.
4. **Crise financement public culture** (baisse budgets État/collectivités) → report sur financements privés / mécénat / crowdfunding → renforce l'intérêt de l'axe "financement 360°" vs "subvention seule".
5. **Transparence des attributions** (open data) : potentiel scraping d'archives de dossiers réussis.

→ **Action** : bâtir sur (1) et (4). Cela justifie la vision "prospecting élargie" déjà dans MEMORY.

## 4. Trois tiers de non-clients

### Tier 1 — "Soon-to-be" (proches du marché)

- Artistes qui ont essayé Aides-Territoires ou DRAC et ont **abandonné** face à la complexité
- Membres d'assos qui utilisent SubventIA collectivement mais n'ont rien pour leur carrière **individuelle**
- Usagers de newsletters spécialisées (Profession Spectacle, devenir.art) qui veulent plus de personnalisation

**Déblocage** : UX conversationnelle + matching multi-rôles + prix < 120€/an → conversion directe.

### Tier 2 — "Refusing" (ont refusé consciemment)

- Artistes convaincus que **"les subventions c'est pas pour moi"** (auto-exclusion : pas assez connu, mauvaise région, mauvais statut)
- Artistes signés par agent/label qui pensent que **tout est délégué** mais dont l'agent ne couvre que 10% des opportunités
- Créatifs **hybrides** (musicien-illustrateur) qui ne rentrent dans aucune case des formulaires standards

**Déblocage** : score "adéquation carrière" + anti-angoisse admin + archive de dossiers réussis "qui me ressemblent".

### Tier 3 — "Unexplored" (marchés distants)

- **Créatifs de plateformes** (YouTubers, podcasteurs, streamers, créateurs Substack) : il existe des bourses/aides pour eux, ils n'y cherchent pas
- **Artisans d'art** (céramistes, verriers, relieurs) : marché CNAP/métiers d'art existe, invisible
- **Chercheurs-artistes** (art+science, art+tech) : fonds mixtes recherche/culture
- **Diaspora créative française à l'étranger** : Institut français, programmes internationaux

**Déblocage** : repositionnement "OS du financement créatif" élargit le total addressable market.

### Commonalité transverse trois tiers

Tous partagent : **surcharge informationnelle + absence de guide personnalisé + peur de la légitimité**. La solution ne vend pas des subventions, elle vend **"on s'occupe de te dire quoi viser et quand"**.

## 5. Séquence stratégique

### 5.1 Utilité acheteur — Buyer Utility Map

Matrice 6 leviers × 6 étapes, focus sur blocages actuels du marché.

| Étape → | Découverte | Décision | Préparation | Soumission | Suivi | Post-attribution |
|---|---|---|---|---|---|---|
| Productivité | 🔴 noyade infos | 🔴 pas de tri | 🟡 templates rares | 🟡 portails hétérogènes | 🔴 pas de vue | 🔴 zéro outil |
| Simplicité | 🔴 jargon | 🔴 critères opaques | 🟡 | 🔴 multi-portails | 🔴 | 🔴 |
| Commodité | 🟡 | 🔴 temps perdu | 🟡 | 🔴 délais serrés | 🔴 | 🔴 |
| Réduction risque | 🔴 éligibilité floue | 🔴 | 🟡 | 🟡 | 🔴 | 🟡 |
| Plaisir/image | 🔴 stress | 🔴 | 🔴 fatigue | 🔴 | 🔴 | 🟡 |
| Écologie/valeurs | ⚪ | ⚪ | ⚪ | ⚪ | ⚪ | ⚪ |

🔴 = bloc majeur, 🟡 = moyen, ⚪ = non pertinent.

**Blocs à attaquer prioritairement** (ROI max) :
1. **Découverte** (productivité + simplicité) → cœur de SubventionMatch
2. **Décision** (réduction risque + plaisir) → zone peu investie par concurrents = blue ocean
3. **Suivi + post-attribution** (toute la ligne) → personne ne le fait = blue ocean
4. **Post-attribution** → création de catégorie possible

### 5.2 Prix stratégique

Mapper contre **alternatives**, pas concurrents directs.

| Alternative | Coût (€/an) pour artiste |
|---|---|
| Rien faire | 0 |
| Excel + newsletters | 0 |
| Comptable/assistant à mi-temps | 2 000–8 000 |
| Agent (commission) | 10–20% d'un deal obtenu |
| SubventIA | 120 |
| Instrumentl | ~3 500 |

**Zone stratégique accessible masse** : **60–200€/an** pour tier standard, **400–800€/an** pour tier pro (prospecting élargi), **1500–3000€/an** pour licence institution.

Piste tarifaire :
- **Free** : 3 matchings/mois, pas de PDF, pas de prospecting privé
- **Pro** : 8–15€/mois → matching illimité + PDF + suivi
- **Carrière** : 30–50€/mois → prospecting privé (agents, galeries, labels) + post-attribution + support
- **Institution** : devis (écoles, sociétés d'auteurs, labels)

### 5.3 Coût cible

Budget unitaire à tenir pour scaler sur le tier Pro à 10€/mo (~120€/an = break-even vs SubventIA) :
- **LLM** (DeepSeek via OpenRouter) : déjà optimisé, ~0,01€/match → tolérable si cache agressif
- **Scraping + enrichissement** : amortir via async batching nocturne
- **Infra Railway + Supabase** : linéaire, OK
- **Puppeteer PDF** : coûteux en RAM, générer à la demande uniquement

**Levier coût** : **data en commun avec tous les users** (un enrichissement fait une fois sert tout le monde). Ton architecture le fait déjà (persisté en DB) — bon point.

### 5.4 Adoption — freins

| Frein | Acteur | Mitigation |
|---|---|---|
| "Encore un outil payant" | Artiste | Free tier généreux + preuve rapide de valeur |
| "Je n'ai pas le temps de m'inscrire" | Artiste | Onboarding conversationnel < 3 min |
| "Pourquoi te faire confiance ?" | Artiste | Preuves sociales (archives réussites), témoignages ciblés par discipline |
| "On ne veut pas d'intermédiaire" | Institutions (DRAC, sociétés d'auteurs) | Partenariats, pas substitution. Positionner comme complément. |
| "L'IA va mal conseiller" | Artiste prudent | Transparence : sourcer chaque match, montrer le raisonnement |
| Saisonnalité (gros pic janvier-février DRAC) | Plateforme | Churn élevé si pas de valeur hors saison → module post-attribution + veille continue |

## 6. Catégorie produit — reformulation

### Ancienne formulation (red ocean)

> "SubventionMatch aide les artistes à trouver des subventions grâce à l'IA"

### Nouvelle formulation (blue ocean)

> "SubventionMatch est l'assistant de carrière du créatif indépendant : il repère, priorise et suit toutes les opportunités de financement — publiques, privées et professionnelles — adaptées à ton œuvre et ton moment de carrière."

**Diff critiques** :
- "assistant de carrière" ≠ "moteur de recherche"
- "opportunités de financement" ≠ "subventions" (élargit)
- "publiques, privées, professionnelles" = inclut agents/galeries/labels/éditeurs
- "ton œuvre et ton moment de carrière" = profilage qualitatif, pas admin

## 7. Score actuel et plan 10/10

### Score actuel estimé : **4/10**

Decomposition :
- Value innovation (différenciation + bas coût) : 5/10 (différenciation culture + beta gratuit, mais pas de création de catégorie)
- Courbe Strategy Canvas distincte : 4/10 (similaire à SubventIA sauf niche)
- ERRC complet (4 actions) : 3/10 (raise oui, create pas encore matérialisé, eliminate timide)
- Non-clients convertis : 3/10 (on cible clients existants de SubventIA/DRAC)
- Utilité leap : 5/10 (qualité matching + multi-rôles = marginal, pas 10x)

### Plan pour atteindre 10/10

**Phase 1 — Consolider le edge culture (2026 Q2–Q3)**
1. Matérialiser le "score adéquation carrière" (au-delà éligibilité) → dans `ai-matcher.ts`
2. Onboarding conversationnel (remplacer formulaires longs)
3. Archive anonymisée de dossiers réussis (contenu éditorial + scraping données publiques d'attribution)
4. Module "calendrier annuel personnalisé"

**Phase 2 — Créer la catégorie (2026 Q3–Q4)**
5. Module prospecting privé : labels, agents, galeries, éditeurs (MVP : 30 acteurs par discipline, matching manuel, puis IA)
6. Module post-attribution (rappels rapport, templates bilan)
7. Mode collectif / compagnie

**Phase 3 — Ouvrir B2B2C (2027)**
8. Pilote avec 1 école d'art + 1 société d'auteurs (SGDL ou ADAGP, plus accessible que SACEM)
9. Offre institution (licence pour membres/étudiants)
10. Pivot positioning marketing : "assistant de carrière" pas "matcher de subventions"

**Phase 4 — Verrouiller (2027+)**
11. Communauté privée (Discord) : l'outil + le réseau
12. Data moat : dataset propriétaire de dossiers qualifiés + feedback d'attribution
13. Partenariats distribution : Institut français (diaspora), Creative Europe desks

### Indicateurs de réussite 10/10

- Un utilisateur ne peut pas nous comparer à SubventIA sans expliquer en 2 phrases que c'est "autre chose"
- NPS >50 dans la communauté artiste indépendant
- 40%+ des conversions payantes viennent du tier "prospecting" (preuve qu'on n'est plus un matcher de subventions)
- 1 partenariat institution majeur (école ou société d'auteurs) signé

## 8. Actions immédiates (≤ 30 jours)

1. **Valider le repositionnement** par 5–10 entretiens courts avec des artistes indépendants de disciplines variées (cinéaste, écrivain, plasticien, musicien, auteur BD). Questions : pain points réels, perception de SubventIA, volonté de payer pour prospecting privé.
2. **Prototyper le score "adéquation carrière"** dans `server/ai-matcher.ts` : ajouter un axe qualitatif au-delà du quality gate (ex: "sert-il l'émergence ? la confirmation ? la consolidation ?").
3. **Cartographier 30 acteurs de prospecting privé** par discipline prioritaire (musique ou écriture) → test de feasibilité du module prospecting.
4. **Rédiger la landing page v2** avec le nouveau pitch ("assistant de carrière créatif"). Tester conversion.
5. **Archiver cette stratégie dans `.planning/strategy/`** (fait) et la relier à ROADMAP si GSD utilisé.

## 9. Risques et anti-patterns à surveiller

- **Trop d'ambition scope** : le module prospecting privé est un produit en soi. Ne pas le lancer avant d'avoir solidifié le cœur subvention culture.
- **Imiter SubventIA par réflexe** : résister à l'envie d'ajouter "génération de docs" juste parce qu'ils le font. Notre différenciation = matching + suivi, pas rédaction.
- **Surestimer la valeur de l'IA générative** : elle va devenir commodity. Le moat est la **data propriétaire** (dossiers qualifiés, historique attribution, feedback artiste) et la **communauté**.
- **Vouloir être exhaustif** : rester niche qualifiée. 500 dispositifs bien matchés > 11 000 mal matchés.
- **Sauter le B2B2C** : c'est un canal de distribution puissant mais cycle de vente long. Ne pas en faire dépendre la trajectoire B2C.
